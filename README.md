# Outfit Planner
Web de gestión de outfits y closet virtual.

## Stack
- **Backend:** Node.js + Express.js
- **Base de datos:** PostgreSQL
- **Autenticación:** JWT
- **Documentación:** Swagger
- **Frontend:** React + Vite

## Instalación
```bash
# Clonar el repo
git clone https://github.com/annetryoshka/Outfit-Planner.git

# Backend
cd backend
npm install

# Frontend
cd frontend
npm install
```

## Variables de entorno
Copia `.env.example` a `.env` y llena los valores.
